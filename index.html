Validation Rules

The tool checks each coordinate value against geographic boundaries and data type requirements.

LAT
Latitude Range
Must be between -90 and +90
Represents North/South position on Earth
LON
Longitude Range
Must be between -180 and +180
Represents East/West position on Earth
NUM
Numeric Values Only
Text, symbols, or empty cells are flagged as invalid

Common Issues Detected

The tool identifies and flags these common coordinate problems in health facility data:

Empty values - Missing coordinate data
Text values - "N/A", "TBD", names, etc.
Out of range - Values like 999, -999
Swapped coordinates - Lat/Lon reversed
Valid pairs - Both values within range

Note: Invalid values are cleared (set to empty) while preserving all rows. This allows you to later fill in correct coordinates without losing other facility data.
